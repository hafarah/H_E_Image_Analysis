# H_E_Image_Analysis
Running H&amp;E tissue segmentation model on pancreas images

Whole slide scan done on Axioscan-Slide scanner at 20x.
Images exported from Zen Lite 3.1 from .czi to ome.tiff: Downsampled to 10x and converted to 8 bit
Annotations done using QuPath, 3 annotations per tissue type per image. 
Tissue types: Exocrine, Endocrine, Adipose, Stroma, Ignore/Background, Fold
